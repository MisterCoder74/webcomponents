SYNTAX:
<image-card title="(text)" image="(path)" bordercolor="(color code or name)" textcolor="(color code or name)"
align="(justification)">
(text here)
</image-card>

The element can contain also HTML content between opening and closing tags.


The ALIGN attribute can use the following values: left, right, center, justify
