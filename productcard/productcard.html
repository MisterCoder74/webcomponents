PRODUCT-CARD ELEMENT

This beautiful ritratto is sold by owner for the ridiculous amount of $600.
SYNTAX:
<product-card productname="(text)" bgcolor="(color code or name)" bordercolor="(color code or name)" textcolor="(color code or name)"
align="(justification)" image="(img path)">
(text here)
</product-card>

The element can contain also HTML content between opening and closing tags.


The ALIGN attribute can use the following values: left, right, center, justify
