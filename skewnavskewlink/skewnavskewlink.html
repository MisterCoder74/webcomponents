SKEW-NAV & SKEW-LINK ELEMENTS

Link 1 Link 1 Link 1

SYNTAX:
<skew-nav bgcolor="(color code or name)" textcolor="(color code or name)"  align="(justification)">
 <skew-link url="#" textcolor="(color code or name)" hovercolor="(color code or name)" skewcolor="(color code or name)">
 (text link here) </skew-link>
</skew-nav>

The element can contain also HTML content between opening and closing tags.
WARNING: For Navbar links it is mandatory to use the SKEW LINK component instead of A.
The ALIGN attribute can use the following values: start, center, end
